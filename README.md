# Busca em Profundidade (DFS) - Mapa da Romênia

Implementação de **Busca em Profundidade Recursiva (DFS)** com visualização animada interativa no mapa da Romênia, clássico problema de IA.

![Demo GIF](demo.gif)
*(Execute o código para ver a animação em tempo real)*

##  Funcionalidades

-  Busca DFS recursiva completa
-  Visualização animada passo a passo
-  Cálculo de custo acumulado em tempo real
-  Contador de nós expandidos
-  Mapa geográfico realista da Romênia
-  Destaque visual do caminho atual

##  Instalação

```bash
pip install networkx matplotlib
```

##  Uso Rápido

```python
python dfs_romenia.py
```

A animação inicia automaticamente de **Arad → Neamt**!
